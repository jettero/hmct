About

Written by Paul Miller.

This app is pretty much just a caching GUI for the http;//hiveminder.com/ website. They encourage building applications like this by offering a very nice self-discovering ReSTful API.

Bugs and feature requests should be reported via HMCT-feeedback@voltar.org — this is a spam filtered address that simply feeds the author's hiveminder task queue.

The entire source for this app can be found there as well:

http://github.com/jettero/hmct
