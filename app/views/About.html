About

Written by Paul Miller. This instance was built on ???.

This app is pretty much just a caching GUI for the http://hiveminder.com/ website. They encourage building applications like this by offering a very nice self-discovering ReSTful API.

Bugs and feature requests can be reported via HMCT-feeedback — this is a spam filtered email address that simply feeds my hiveminder task queue.

You may also report bugs and request features by using the github issue tracker:

http://github.com/jettero/hmct/issues

The entire source for this app can be found on github:

http://github.com/jettero/hmct

Some unreleased, release candidate and historical builds can be found on the gh-pages:

http://jettero.github.com/hmct
